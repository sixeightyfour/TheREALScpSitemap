# The (Unofficial) *Real* SCP Sitemap

An attempt to map the locations of:

* Where every mainlist -EN SCP was found
* Where every -EN tale takes place
* Where every -EN site is located

Very much a WIP. Ignore for now.

Love, Danjon :D

## Workflow
### Stage 1
Locations are being taken directly from articles and entered into a GeoJSON.
At this stage, articles with the following locations are being labeled:
* Articles where the SCP was discovered or recovered in a specific location (Ex. SCP-031)
* Articles where the SCP was written to have been discovered or recovered at the county level (or international equivalent) or lower.
